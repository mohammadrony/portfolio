# Data Grip

- Open [Installation guide](https://www.jetbrains.com/help/datagrip/installation-guide.html) > Check System requirements > Install using the Toolbox App > Select Operating System > Download the installer > [Toolbox app](https://www.jetbrains.com/toolbox-app/) > Select Binary > Download.
- Open Toolbox app > Install > Tools > DataGrip > Install > Open DataGrip.
