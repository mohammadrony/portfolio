# PM2

## Prerequisites

[Install Nodejs and NPM](../packages/node-npm.md)

## Installation

```sh
npm install -g pm2
```

## Commands

Application list

```sh
pm2 list
```

Start application

```sh
pm2 start app.js
```

Auto restart service

```sh
pm2 startup
```

```sh
sudo env PATH=$PATH:...
```

```sh
pm2 save
```
