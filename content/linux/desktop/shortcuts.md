# Keyboard Shortcuts

Ubuntu

| Action                                              | Key map               |
|-----------------------------------------------------|-----------------------|
| Launchers > Launch web browser                      | Alt + F               |
| Navigation > Move window one monitor to the left    | Super + Left          |
| Navigation > Move window one monitor to the right   | Super + Right         |
| Navigation > Move window one workspace to the left  | Alt + Super + Left    |
| Navigation > Move window one workspace to the right | Alt + Super + Right   |
| Navigation > Move/Switch to workspace on the left   | Ctrl + Super + Left   |
| Navigation > Move/Switch to workspace on the right  | Ctrl + Super + Right  |
| Navigation > Switch to workspace 1                  | Alt + Super + 1       |
| Navigation > Switch to workspace 2                  | Alt + Super + 2       |
| Navigation > Switch to workspace 3                  | Alt + Super + 3       |
| System > Show the overview                          | Super + S             |
| Window > Hide window                                | Alt + D               |
| Window > Toggle maximization state                  | Super + Up            |
| Window > Restore window                             | Super + Down          |
