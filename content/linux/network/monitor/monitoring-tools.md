# Monitoring tools

System monitoring tool

```sh
nmon
```

System resource statistics

```sh
dstat
```

Display current network usage

```sh
nload
```

Bandwidth monitor

```sh
bmon
```

TCP connections on the network

```sh
tcptrack
sudo tcptrack -i ens3
```

Network load monitor

```sh
slurm
slurm -i ens3
```
