# Change Modification

## Restore

Restore file in current branch

```sh
git restore *.file
```

```sh
git restore ./path/to/file
```

## Stash

Save changes

```sh
git stash
```

Apply latest stash

```sh
git stash apply
git stash drop
```

```sh
git stash pop
```

Clear all saved copy

```sh
git stash list
git stash clear
```
