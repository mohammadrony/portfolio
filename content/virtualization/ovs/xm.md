# XM

## Commands

```bash
xm
```

VM status

```bash
xm list
```

```bash
xm uptime
```

VM management

```bash
xm create <file>
```

```bash
xm domname <id>
```

```bash
xm destroy <id>
```

```bash
xm reboot <id>
```

```bash
xm console <id>
# Ctrl+] to exit
```

VNC viewer

```bash
xm vncviewer -h
```
