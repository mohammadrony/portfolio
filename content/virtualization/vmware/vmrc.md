# VMware Remote Console

[Installation of the VMware Remote Console Application](https://knowledge.broadcom.com/external/article/368995/)

- Create an Account in [Broadcom](https://support.broadcom.com/)
- Follow the [steps](https://knowledge.broadcom.com/external/article/368995/) to download `vmrc` installer.
- Installation for Linux
  - Execute `sudo ./VMware-Remote-Console-xxx.x86_64.bundle` from CLI.
- Installation for Windows
  - Extract zip and run `VMware-VMRC-xxx.exe` file.
