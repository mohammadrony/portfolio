# VMWare Workstation Pro

- Create an Account in [Broadcom](https://support.broadcom.com/).
- [Download](https://support.broadcom.com/group/ecx/downloads) VMware Workstation Pro for Linux.

List vmware products

```bash
sudo vmware-installer -l
```

Uninstall vmrc

```bash
sudo vmware-installer -u vmware-vmrc
```

Install Workstation

```bash
chmod +x VMware-Workstation-Full-xxx.x86_64.bundle
```

```bash
sudo ./VMware-Workstation-Full-xxx.x86_64.bundle
```
