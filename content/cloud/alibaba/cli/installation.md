# Alibaba Cloud CLI Installation

[Alibaba Cloud CLI Overview](https://www.alibabacloud.com/help/en/alibaba-cloud-cli)

```bash
wget https://aliyuncli.alicdn.com/aliyun-cli-linux-latest-amd64.tgz -O aliyun-cli.tgz
tar xzvf aliyun-cli.tgz
sudo mv aliyun /usr/local/bin/
rm aliyun-cli.tgz
```

Commands

```bash
aliyun
```
