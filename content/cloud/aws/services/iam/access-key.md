# Access Key

## Generate Access and Secret Key

- Goto [IAM Dashboard](https://console.aws.amazon.com/iam/home) >
- Users > Select User >
- Security credentials > Access keys > Create access key >
- Select Command Line Interface (CLI) >
- Confirm to create an access key > Next >
- Enter Description > Create access key >
- Copy Access key and Secret access key >
- Download .csv file > Done.

## Deactivate Access keys

- Goto [IAM Dashboard](https://console.aws.amazon.com/iam/home) >
- Users > Select User >
- Security credentials > Select Access keys > Actions > Deactivate > Deactivate.

## Delete Access keys

- Goto [IAM Dashboard](https://console.aws.amazon.com/iam/home) >
- Users > Select User >
- Security credentials > Select Access keys > Actions > Delete > Enter access key ID > Delete.
