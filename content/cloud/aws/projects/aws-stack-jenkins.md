# AWS Stack with Jenkins

Learning project from **BJIT**.

![AWS deployment architecture diagram](../architecture/aws-stack-diagram.png)

## Used Service

- EC2
- RDS
- Route53
- ALB
- ACM

## Service Setup

- Create VPC, subnets, and route tables
- Create EC2 instance, RDS database
- Create SSL/TLS Certificate
- Create Load balancer and Auto scaling group
- Create Route 53 domain
