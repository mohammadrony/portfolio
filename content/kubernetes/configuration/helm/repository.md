# Chart Repository

## Find repository

Search repository

```bash
helm search repo
```

Chart repository chart

```bash
helm search repo <chart>
```

```bash
helm search repo -l <chart>
```

## Add repo

```bash
helm repo add stable https://charts.helm.sh/stable
```

```bash
helm repo add <name> <repository>
```

Update Helm repository

```bash
helm repo update
```
