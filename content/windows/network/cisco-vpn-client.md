# Cisco VPN Client

## Cisco AnyConnect Secure Mobility Client

Installation

- Open [Firewall.cx](https://www.firewall.cx/downloads/cisco-tools-a-applications/cisco-anyconnect-secure-mobility-client-v4-9-0195.html) from Browser > [Download Cisco AnyConnect Secure Mobility Client](https://www.firewall.cx/component/edocman/cisco-anyconnect-secure-mobility-client-v4-9-0195/download.html) > Extract File > Start Installer file (.msi).

Connect to VPN

- Open **Cisco AnyConnect Secure Mobility Client** from Start menu > Enter VPN Address > Connect > Disable **Block connections to untrusted servers** > Connect Anyway > Enter Username > Enter Password > Connect.

## Microsoft AnyConnect

Installation

1. Install [AnyConnect - Microsoft Store](https://www.microsoft.com/store/productId/9WZDNCRDJ8LH)
2. Open AnyConnect > Settings > Block Untrusted Servers > OFF.

Connect to VPN (Failed)

1. Open VPN settings > Add VPN > VPN provider > AnyConnect > Connection name > Server name or address > Select Remember my sign-in info > Save.
2. Select VPN to connect > Enter Username > Enter Password > Next.
